Список полезных для web разработчика сайтов

  • jsfiddle.net

    Сайт для написания кода на html, css, js. Можно делиться результатом, отправив ссылку на своё решение.

  • codepen.io

    Сайт для написания кода на html, css, js. Можно просматривать готовые решения более профессиональных ребят.

  • validator.w3.org

    Сайт для проверки валидности html кода от создателей этого стандарта.

  • css_websafe_fonts.asp

    Сайт, где можно посмотреть возможные комбинации шрифтов для свойства font-family. А если перейти на главную откроется w3school, которую тоже можно поизучать.

  • caniuse.com

    Сайт, где можно посмотреть какие браузеры поддерживают тег или css-свойство.

  • getbootstrap.com

    Сайт откуда можно скачать самый популярный фреймворк для fronend'a. А также посмотреть как им пользоваться.

  • jquery.com

    Сайт, откуда можно скачать jQuery (необхоимо для работы bootstrap).

P.S. Этот список предназначен в первую очередь для меня самого, а сайт будет использоваться для практики web программирования.
