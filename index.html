Список полезных для web разработчика сайтов

  • jsfiddle.net - cайт для написания кода на html, css, js. Можно делиться результатом, отправив ссылку на своё решение.
  • codepen.io - cайт для написания кода на html, css, js. Можно просматривать готовые решения более профессиональных ребят.
  • validator.w3.org - сайт для проверки валидности html кода от создателей этого стандарта.
  • css_websafe_fonts.asp - сайт, где можно посмотреть возможные комбинации шрифтов для свойства font-family. А если перейти на главную откроется w3school, которую тоже можно поизучать.

P.S. Этот список предназначен в первую очередь для меня самого, а сайт будет использоваться для практики web программирования.
